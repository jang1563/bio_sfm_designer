# Curated Results

This directory contains compact scientific release evidence only.

- `m6d_w2b_target_adaptive_certification_report.json` is the terminal exact-LTT
  W2b report replayed by the public tests.
- `m6d_w2c_target_selection.json` records deterministic, label-blind selection
  of eight prospective W2c targets. It is not a compute or outcome artifact.

Scheduler metadata, submission receipts, logs, raw model state, private paths,
and session/operator ledgers are excluded from the public snapshot.
